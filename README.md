# Eth Unit Converter

A minimal web tool for converting between Ethereum denomination units — wei, kwei, mwei, gwei, szabo, finney, and ether.

Built with [Next.js](https://nextjs.org), [Chakra UI](https://chakra-ui.com), and [BigNumber.js](https://mikemcl.github.io/bignumber.js/) for precise arithmetic.

## Getting Started

```bash
# Install dependencies
yarn install

# Run development server
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Unit Reference

| Unit   | Wei                        |
| ------ | -------------------------- |
| wei    | 1                          |
| kwei   | 1,000                      |
| mwei   | 1,000,000                  |
| gwei   | 1,000,000,000              |
| szabo  | 1,000,000,000,000          |
| finney | 1,000,000,000,000,000      |
| ether  | 1,000,000,000,000,000,000  |
